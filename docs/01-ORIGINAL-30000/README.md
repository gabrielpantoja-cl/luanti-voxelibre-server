# Wetlands (Puerto 30000)

Mundo principal: creativo, educativo, apto para niños. NPCs no-violentos, animales, música, PvP arena opt-in.

| Aspecto | Valor |
|---------|-------|
| Puerto | 30000/UDP |
| Juego | VoxeLibre (mineclone2) |
| Mapgen | v7 |
| Spawn | `0,15,0` |
| Creativo | Sí |
| Daño | Sí (mobs hostiles de noche) |
| PvP | Solo en arena (mod `pvp_arena`) |
| Mods clave | `wetlands_npcs`, `wetlands_music`, `wetlands_no_creeper`, `wetlands_lastpos`, `voxelibre_protection`, `pvp_arena` |
